TLV Exam Calendar

Use the tick box on each day to show whether the candidate is in. Fixed breaks are always removed: 11:10 to 11:25, 13:35 to 14:15, and 15:20 to 15:25. A 10 minute buffer is also added at the start of each day and after each break. Each exam also starts with one reading session before the first main task.

Tasks

Choose one task, then place it into the calendar, or auto-place across all four weeks.

Auto-place uses four weeks and starts each new task on a later day than the previous task finished.
