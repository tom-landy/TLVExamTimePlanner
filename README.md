# Weekly Exam Task Plotter

Small web app for the core scheduling problem:

- enter the student's available hours for each weekday
- enter the exam tasks and how long each one takes
- plot those tasks into a simple weekly calendar
- print the result

## Run locally

```sh
cd /Users/tomlandy/Documents/Playground
npm start
```

Then open [http://localhost:3000](http://localhost:3000).

## Deploy on Render

The app is ready for a Render `Web Service` with:

- Build Command: `npm install`
- Start Command: `npm start`

You can use the included [render.yaml](/Users/tomlandy/Documents/Playground/render.yaml).
